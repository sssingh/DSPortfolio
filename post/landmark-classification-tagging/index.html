PROJECT: Landmarks Classification & Tagging

October 22, 2021

Here we tackle a multi-label-classification problem. Photo sharing and photo storage services like to have location metadata for each photo that is uploaded. If no location metadata for an image is available, one way to infer the location is to detect and classify a discernible landmark in the image manually which is a very big ask if number of images are large. In this project we’ll try to address this problem by building Neural Network (NN) based models to automatically predict the location of the image based on any landmarks depicted in the image.

Our objective here is to build a Deep Neural Network based model that’d accept any user-supplied image as input and suggest the top k most relevant landmarks from 50 possible landmarks from across the world.

The neural-network will be build using convolutional-neural-network (CNN) which is the de-facto technology for computer-vision tasks now days. We experiment with two different strategies…

  1. Build the CNN neural-network from Scratch and train it. No matter deep we go with this option our network will still be relatively shallow compared to state-of-the-art CNN networks in the wild. Hence here we aim to attain a test accuracy of at least 30%. At first glance an accuracy of 30% may appear to be very low but its way better than random guessing, which would provide an accuracy of just 2% since we have 50 different landmarks classes in the dataset.
  2. Build the CNN neural-network using ImageNet trained state-of-the-art models through transfer-learning. Here we aim to attain a test accuracy of at least 60% which is pretty good given the complex nature of this task.

  • Prediction accuracy is used as the model’s performance metric on the test-set
  • The CNN neural-network from Scratch is build using 5 Convolutional layers followed by a fully-connected layer finally producing a LogSoftMax output. Network is then trained and validated for 15 epochs using Negative Log Likelyhood Loss function and Adam optimizer with a learning rate of 0.001. With this setup we could achive an accuracy of 38%
  • The CNN neural-network through transfer-learning is built using VGG19 pre-trained model on ImageNet. The classifier layer is replaced with a custom fully-connected classifier finally producing a LogSoftMax output. All network weights are frozen except for custom classifier weights. Network is then trained and validated for 10 epochs using Negative Log Likelyhood Loss function and Adam optimizer with a learning rate of 0.001. With this setup we could achive an accuracy of 63%
