Project

SVHN and Celebrity Face Generation using DC-GAN

⚡Image Generation
⚡Deep Convolutional GAN (DC-GAN)
⚡Convolution and Transpose Convolution Layers
⚡SVHN & CelebA
⚡PyTorch

DC-GAN is an extension and builds on the same idea as GAN networks. Like GAN, DC-GAN is also composed of two competing networks, the Discriminator that classifies images as real or fake and the Generator that learns from the actual dataset (MNIST, SVHN, CelebA, etc.) and generates the counterfeit images. However, in the case of DC-GAN, instead of simple fully-connected Linear layers, Convolution and Transpose Convolution layers are used for network composition.

  • Refer MNIST Digit Generation Using GAN project for a detailed discussion on GAN network.

  • Have a look at DC-GAN Original Paper for in depth discussion on the subject.

Our goal in this project is to build two DC-GAN networks. The first network will train over the Street View House Numbers (SVHN) dataset so that the network learns to generate synthetic SVHN images. The second network will train over the CelebA dataset to learn to generate imaginary celebrity faces.

Project

MNIST Digit Generation using GAN

⚡Image Generation
⚡Generative Adverserial Network (GAN)
⚡Fully Connected Neural Layers
⚡MNIST
⚡PyTorch

GAN is a neural-network architecture used for Generative Modeling. The Generative Models are used to generate new ‘made-up/fake’ examples that’d appear to have come from the authentic distribution of samples. GANs are trained using two separate networks, a Generator, and a Discriminator, competing like adversaries. The Discriminator’s job is to detect the actual image from the dataset vs. the fake image generated by the trained Generator.

Have a look at GAN Original Paper for more details.

Our goal in this project is to build a GAN and train it over the MNIST dataset so that the network learns to generate fake MNIST digit images that would appear to have come from the real MNIST dataset. Still, in reality, these images are not part of the MNIST dataset.

TV Script Generation using RNN

⚡Text Generation
⚡Recurrent Neural Network (RNN)
⚡Gated Recurrent Unit (GRU)
⚡PyTorch

Text Generation is a subfield of Natural Language Processing (NLP). In this project, we train a neural network to learn the underlying structure and pattern in a given text corpus (tv script in our case) and then generate a never seen before (but fake) tv script. There are many real-life applications of text generating models, such as:

  • Product reports describing the features of a new product
  • Financial reports
  • Personalized emails.
  • Quickly generating unique marketing content targeting a unique customer.
Project

Landmarks Classification & Tagging

⚡Multi Label Image Classification
⚡Cutsom CNN
⚡Transfer Learning CNN
⚡PyTorch

Here we tackle a multi-label-classification problem. Photo sharing and photo storage services like to have location metadata for each uploaded photo. However, suppose no location metadata for an image is available. In that case, one way to infer the location is to detect and classify a discernible landmark in the shot manually, which is very tough if the number of images is significant. In this project, we’ll try to address this problem by building a `Neural Network (NN) based model to automatically predict the location of the image based on any landmarks depicted in the picture.

Project

Fashion-MNIST Classification

⚡Multi Label Image Classification
⚡Cutsom Fully Connected NN
⚡Fashion MNIST
⚡PyTorch

Fashion-MNIST Classification is again a multi-label classification task. This project is very similar to the MNIST Digit Classification project. However, instead of hand-written digits, we classify apparel images (clothes, shoes, bags, etc.) into ten distinct classes. The main difference is that the Fashion-MNIST dataset is more complex than the MNIST dataset. Therefore, getting an accuracy near 95% or above by using fully-connected neural-network layers (i.e. without using the Convolution layer) would be challenging.

Hand Written Digit Classification

⚡Multi Label Image Classification
⚡Cutsom Fully Connected NN
⚡MNIST
⚡PyTorch

In this project, we deal with a multi-label classification problem. The MNIST dataset is widely used in academia and industry for training and testing machine-learning models. Therefore, MNIST dataset classification is possibly one of the very first tasks an aspiring ML/DL learner encounters. I think of this as the “Hello World” for classification tasks to experiment with various ML and DL models.

Our objective is to build a neural-network classifier to discriminate between digits 0 to 9 in the MNIST dataset.
